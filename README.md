# goit-advancedjs-hw-02

**Module: Advanced JavaScript and TypeScript**

## Description

Asynchronous JavaScript in practice — a countdown timer with a date picker and a promise-based notification generator.

## Tasks

- **Countdown Timer** — user picks a future date via flatpickr; the timer counts down days, hours, minutes and seconds and stops at `00:00:00:00`.
- **Promise Generator** — a form that creates a promise with a custom delay; resolves or rejects based on the selected state and shows a toast notification via iziToast.

## Stack

- Vanilla JavaScript (ES6+)
- Vite
- [flatpickr](https://flatpickr.js.org/)
- [iziToast](https://izitoast.marcelodolza.com/)
